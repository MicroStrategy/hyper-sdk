Enable / Disable Cards

Enable selected cards during initialization or call APIs to enable/disable cards at any time.

Option 1: specify the cards to enable in mstrHyper.start

Enable cards during initialization

By default, Hyper SDK enables all the Hyper Cards that are available to the end-user. If you want to enable a selected list of Hyper Cards, you can list them in the initialization settings.

Option 2: call enable/disable cards APIs

Enable Cards API

After the Hyper SDK is initialized, you may enable any cards at any time with the mstrHyper.enableCards API.

Disable Cards API

After the Hyper SDK is initialized, you may disable any cards at any time by calling the mstrHyper.disableCards API.
